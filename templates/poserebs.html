MOTIONMATE
HOW WE IMPLEMENTED MOTIONMATE

  • Here is a high-level algorithm for counting reps using Mediapipe:
  • Import the necessary packages and load the Mediapipe Pose model
  • Load the video or webcam stream that you want to analyze
    Initialize variables to keep track of the pose landmarks and the number of reps
  • Define a function to calculate the angle between three landmarks
  • Process each frame of the video stream and detect the pose landmarks using Mediapipe
  • Check if the pose landmarks for the exercise you want to count reps for are present
  • Calculate the angle between the relevant landmarks and count the reps(landmarks can be of Elbow,wrist and shoulder)
  • Display the video stream with the number of reps
  • Use variables with condition such that it doesn't count on single hand reps
  • Note that this algorithm is a high-level overview and may require some additional steps depending on the specifics of your use case, such as choosing the relevant landmarks for the exercise you want to count reps for, defining the angles for the start and end of a rep, and tuning the values for those angles to match the movements of the exercise. Also note that the algorithm assumes a single person is present in the video stream.
