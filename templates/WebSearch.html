HOW WE IMPLEMENTED IMAGESLEUTH

  • Process each frame of the video or image:
  • Refer to algorithms of OBJECTMAPPER for detection of objects and stores labels of detected object in list for web searching
  • Import the necessary libraries
    Initialize a variable "topic_search" with the search topic
  • eplace any spaces in "topic_search" with "+"
  • Initialize a new instance of the Edge web driver using the "msedgedriver.exe" file
  • Construct a URL by concatenating "https://www.bing.com/search?q=" with "topic_search" and "&start" with the string representation of "i"
  • Call the "browser.get()" method with the URL constructed in above step
  • Store the return value of "browser.get()" in a variable "elements" this will open search in browser
